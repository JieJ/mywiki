用Jekyll搭建个人博客

一、安装部分

1. 安装Ruby

因为Jekyll基于Ruby开发

brew install ruby

2. 安装Gem

什么是Gem？

RubyGems是一个包管理框架，提供了ruby社区gem的托管服务，用于方便地下载、安装和使用ruby软件包。ruby软件包被称为gem，包含了ruby应用或库。

一般Mac自带，没有的话去官网下载安装

3. 安装Bundle

依赖问题，具体未知

安装完成，jekyll -v check 版本

一、开始使用

1. 本地博客

jekyll new blog  # 新建博客目录
cd blog
jekyll serve # 启动博客

本地访问地址：https:localhost:4000

小插曲: jekyll serve报错
Error: Address already in use - bind(2)
原因是上次关闭时使用了control+z，应该是control+c；找到相应进程杀死，问题就解决啦。
